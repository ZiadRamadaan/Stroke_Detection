How the System Works

Predictive Healthcare Analytics. Stroke is a leading cause of death and disability. This project leverages advanced Machine Learning to evaluate medical records and provide accurate, real-time stroke risk predictions.

15 Clinical Features

Analyzes key indicators like BMI, Glucose levels, Hypertension, and smoking status.

High Accuracy

Achieved 84% Accuracy & 82% ROC-AUC using an optimized Ensemble Model.

Data Balancing

Implemented BorderlineSMOTE technique to handle severe class imbalance effectively.

Real-time Inference

Instantaneous risk scoring interface designed for clinical decision support.
